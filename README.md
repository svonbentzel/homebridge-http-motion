# homebridge-http-motion

## Installation
(Login as root with ```sudo su```)
```bash
git clone https://github.com/svonbentzel/homebridge-http-motion.git
cd homebridge-http-motion
npm install -g
```

## Configuration
```json
  "accessories": [
    {
        "accessory": "http-motion-switch",
        "name": "Remote-Switch",
        "port": 18089          
    }
      ]
```
