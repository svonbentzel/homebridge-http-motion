# homebridge-http-motion

## Installation
Login as root
```bash
git clone https://github.com/svonbentzel/homebridge-http-motion.git
cd homebridge-http-motion
npm install -g
```

## Config
```json
  "accessories": [
    {
        "accessory": "http-motion-sensor",
        "name": "Remote-Switch",
        "port": 18089          
    }
      ]
```
## Usage
http://localhost:18089
http://xxx.xxx.xxx.xxx.18089
